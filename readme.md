## FOAD CEPPIC – DWWM 2025
Portfolio – HTML / CSS / JavaScript

Ce projet a été réalisé dans le cadre de la formation FOAD CEPPIC – DWWM 2025.
Il s’agit d’un site portfolio responsive développé à partir des maquettes mobile et desktop fournies.

## Fonctionnalités

• Menu sticky visible lors du défilement

• Effets parallax sur les images principales

• Sous-menu Portfolio affichant les miniatures

• Modale d’images avec slider (navigation et titre dynamique)

• Galerie cliquable

• Menu hamburger sur mobile

• Mise en page entièrement responsive

• Formulaire de contact

## Technologies
• HTML5
• CSS3 (Flexbox, Grid, parallax, responsive)
• JavaScript vanilla (interactions et slider)
• Icônes Font Awesome
• Police locale Lato-Regular.ttf
